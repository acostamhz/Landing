# Grupo Caishen

Landing page corporativa desarrollada para Grupo Caishen.

---

## Stack Tecnológico

Este proyecto está construido con las siguientes tecnologías:

- Next.js
- React
- TypeScript
- Tailwind CSS
- Framer Motion
- shadcn/ui
- Radix UI
- Lucide Icons

---

## Características

- Arquitectura moderna basada en App Router
- Diseño responsive
- Componentes reutilizables
- Animaciones fluidas
- Estructura escalable
- Optimización de rendimiento
- Tipografía Geist
- Sistema de diseño modular
